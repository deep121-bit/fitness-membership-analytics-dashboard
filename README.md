# 🏋️ Fitness Membership Analytics Dashboard

An interactive **Business Intelligence Dashboard** built using **Microsoft Power BI** to analyze fitness membership performance, revenue trends, customer demographics, and member engagement.

---

## 📌 Project Overview

The Fitness Membership Analytics Dashboard was developed to transform raw membership data into meaningful business insights. This dashboard enables gym management to monitor KPIs, revenue, membership plans, customer behavior, and service utilization through interactive visualizations.

---

## 🎯 Business Objectives

- Monitor membership performance
- Track revenue trends
- Analyze customer demographics
- Measure member engagement
- Evaluate additional service usage
- Support data-driven business decisions

---

## 📊 Dashboard Features

- Executive KPI Cards
- Revenue Analysis
- Membership Plan Analysis
- Gender Distribution
- Customer Age Analysis
- Weekly Visit Analysis
- Interactive Slicers
- Dynamic Filtering

---

## 📈 Key KPIs

- Total Members
- Total Revenue
- Average Revenue per Member
- Average Member Age
- Average Weekly Visits
- Average Session Duration

---

## 🛠️ Tech Stack

- Microsoft Power BI
- Power Query
- DAX
- Data Modeling
- CSV Dataset
